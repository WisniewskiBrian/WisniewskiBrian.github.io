Predicting Burned Area of Forest Fires in Northeast Portugal

Description:

This project aimed to predict the burned area of forest fires in the northeast region of Portugal using multiple regression trees. The objective was to fit models that minimize predictive error and identify key variables influencing the burned area.

Technical Methods and Skills Used:

  • Data Exploration: Used histograms and scatter plots to inspect variables, assess skewness, and identify potential explanatory power with respect to the burned area.
  • Log Transformation: Applied log transformation to the burned area variable to address significant skewness and improve model accuracy.
  • Training and Testing Data: Created random subsets for training and testing, ensuring reproducibility by setting a random seed.
  • Model Building: Fitted and plotted five CART regression models using different variables and tuning criteria, focusing on variables such as temperature, Duff Moisture Code (DMC), and Drought Code (DC).
  • Prediction and Validation: Made predictions on testing data, converted predictions to hectare scale, and computed root mean squared prediction error (RMSPE) for model validation.
  • Averaging Predictions: Computed the average of predictions from multiple models to improve overall prediction accuracy and analyzed the effect of different random seeds on model performance.

Outcome:

The project identified temperature and DMC as key variables in predicting the burned area of forest fires. The log-transformed predictions provided more accurate results compared to original data predictions. Averaging predictions from multiple models improved accuracy, though some models produced infinite RMSPE due to extreme prediction errors. Experimentation with different random seeds showed significant variability in model performance, highlighting the importance of comprehensive testing and validation.

Overall, the project demonstrated the effectiveness of regression trees in predicting forest fire impacts and underscored the need for thorough preliminary analysis and variable selection to enhance model accuracy and reliability.
