Machine Learning to Forecast Google Stock Price

Description:

In this project, I explored the application of machine learning models to predict the stock price of Google (GOOG). The objective was to compare the accuracy and robustness of two machine learning regression models: Random Forest Regression and Support Vector Regression (SVR).

Technical Methods and Skills Used:

  • Data Collection and Preprocessing: Collected historical stock data from Yahoo! Finance, including fields such as Date, Open, High, Low, Close, Adj Close, and Volume.
  • Exploratory Data Analysis (EDA): Conducted initial data analysis to understand correlations and patterns in the dataset.
  • Machine Learning Models: Implemented and trained Random Forest Regression and SVR models to forecast stock prices.
  • Cross-Validation: Employed repeated cross-validation techniques to ensure robust model evaluation.
  • Performance Metrics: Evaluated model performance using Root Mean Squared Error (RMSE) and percentage RMSE.
  • Feature Importance Analysis: Analyzed feature importance in the Random Forest model, addressing challenges related to high collinearity among price variables.

Outcome:

The Random Forest Regression model outperformed the SVR model in terms of accuracy and robustness, as indicated by lower RMSE and percentage RMSE values. This project demonstrates my ability to apply advanced machine learning techniques and perform rigorous model evaluation.
