Redesigning OSU's IDP Calculator with DataSense test

Description:

The Integrated Data Protection (IDP) project involved redesigning Ohio State University's IDP Calculator to improve its usability and user experience. The goal was to create a tool that helps researchers and university staff understand and comply with data protection policies by providing a more intuitive and informative interface.

Technical Methods and Skills Used:

  • User Research and Analysis: Conducted user research to gather insights on current user experiences, pain points, and needs. This involved surveys, interviews, and usability testing.
  • UI/UX Design: Applied user-centered design principles to create wireframes, mockups, and prototypes of the new IDP Calculator interface.
  • Data Categorization and Mapping: Simplified data categorization and mapping processes to make it easier for users to understand and apply data protection requirements.
  • Interactive Features: Implemented interactive features such as tooltips, help icons, and customizable views to enhance user engagement and understanding.
  • Feedback and Iteration: Collected feedback from user testing sessions and iteratively improved the design based on user input.

Outcome:

The redesigned IDP Calculator, named DataSense, was well-received by users during testing. Key improvements included a more intuitive interface, better data categorization, and enhanced interactive features. Users appreciated the clear and informative layout, which made it easier to navigate and understand data protection requirements. Recommendations for further improvements included adding labels to icons, refining the autocomplete feature, and providing pop-up text hints for key elements.
